Ico graphing library Documentation

Ico is a graphing library based on Raphaël and Prototype.js. It is originally developed by Alex Young, but this version was mostly developed by Kilian Valkhof.

Ico's basic principles are to provide good, clean looking graphs inspired by work of Stephen Few and Edward Tufte.

Basics

Ico graphs are initiated via this function call:

   var foo = new Ico.GraphType($(element), data, options);

$(element) is a prototype id selector, data is an object or array (depending on the graph) and options is optional and an object.

The base graph options are applicable to all graphs except for the sparklines and sparkbars.

Dependencies

  • Raphaël 1.0+
  • Prototype.js 1.6+

Base graph options:

width
Specify the width of the element (number)
Default value: width of $(element)
height
Specify the height of the element (number)
Default value: height of $(element)
labels
The labels being displayed on the horizontal axis (array of strings)
Default value: numerical
grid
draw a grid or not (boolean)
Default value: true
plot_padding
Padding at the start and end of the graph (number)
Default value: 10
font_size
The font-size for the axis and hovers (number)
Default value: 10
show_vertical_labels
Show the vertical axis and gridlines (boolean)
Default value: true
show_horizontal_labels
Show the horizontal axis and gridline (boolean)
Default value: true
vertical_label_unit
append a unit to the vertical axis and text in the hover (string)
Default value: empty
colours
Select the colours for the lines or bars (object)
Default value: random colours
background_colour
The background colour of the graph (string, example: "#fff")
Default value: backgroundcolor of $(element), else #fff
label_colour
The colour of the axis, axis values and hovers (string, example: "#fff")
Default value: "#000"
grid_colour
The colour of the gridlines (string, example: "#fff")
Default value: "#ccc"
hover_text_colour
the colour of the text in the hovers (string, example: "#fff")
Default value: "#fff"
meanline
Display a line on the average of all datapoints (object or boolean, example: { 'stroke-width': '2px', stroke: '#000' })
object example: Default value: false
padding_top
The amount of padding above the graph (number)
Default value: 20
draw_axis
show or hide the axis lines (boolean)
Default value: true
datalabels
for line graphs: Add a hover with the text, and append the text to the marker hover, per line (object, example: {workload:'My workload'})
for bar graphs: append a text per bar. (object, example: {workload: ["1","2","5"]})
Default value: empty
hover_colour
colour a bar or line should fade to on hover (string, example: "#fff")
Default value: empty
watermark
Add in an image to be used as watermark. (link to image or image element)
Default value: false
watermark_orientation
where the watermark should be placed. default is the bottom right, can be placed in the center with "middle" (string)
Default value: bottom right
left_padding
Set a standard left spacing regardless of vertical axis label width (number)
Default value: false

Line graphs

Line graphs accept ranges of negative and positive numbers.

Line graph with the default rendering (no options):

   var linegraph = new Ico.LineGraph($('linegraph'),{ workload: [8, 10, 6, 12, 7, 6, 9] });

Line graph with custom options:

   var linegraph2 = new Ico.LineGraph($('linegraph'),{ workload: [8, 10, 6, 12, 7, 6, 9] },
    {
      grid :                  true,
      plot_padding :          0,
      font_size :             11,
      vertical_label_unit :   "hr",
      colours :               {workload: '#4b80b6'},
      background_colour :     "#fff",
      label_colour :          "#444",
      grid_colour :           "#ccf",
      markers :               "value",
      meanline :              { 'stroke-width': '2px', stroke: '#000' },
      draw_axis :             false,
      labels :                ['mon', '', 'wed', '', 'fri', '', 'sun',],
      hide_empty_label_grid : true,
      datalabels :            {workload:'My workload'},
    });

Line graph with multiple lines:

    var linegraph3 = new Ico.LineGraph($('linegraph3'),
      {
      workload:       [8, 10, 6, 12,7, 6, 9],
      your_workload:  [6, 8,  4, 8, 12,6, 2],
      his_workload:   [2, 9,  12,7, 8, 9, 8]
      },
      {
        markers: "circle",
        hover_colour: "#000",
        hover_text_colour: "#fff",
        datalabels: {
          workload:       "My workload",
          your_workload:  "Your workload",
          his_workload:   "His workload"
        }
      });

Line graph with multiple lines and negative values:

    var linegraph4 = new Ico.LineGraph($('linegraph4'),
      {
      a : [8, -10, 6, 12,7, -6, -9],
      b : [6, 8,  -4, 8, -12,6, 2],
      c : [2, 9,  12,7, 8, -9, 8]
      });

.

Line graph options:

markers
Line graph: display markers on the datapoint. (string)
Possible options:
  • "circle" - displays a circle
  • "value" - displays a hover with the value
Default value: false
marker_size
The size of the marker (number)
Default value: 5
hide_empty_label_grid
hide the vertical gridlines for all empty label on the horizontal axis. (boolean)
Default value: false
start_at_zero
If false, Ico will not start drawing untill the value of a line graph becomes more than zero. (boolean)
Default value: true

Area graphs

Area graphs only support positive ranges.

Area graph with the default rendering (no options):

   var areagraph = new Ico.AreaGraph($('areagraph'),{ workload: [8, 10, 6, 12, 7, 6, 9] });

Area graph with custom options:

    var areagraph2 = new Ico.AreaGraph($('areagraph2'),{ workload: [8, 10, 6, 12, 7, 6, 9] },
    {
      grid :                  true,
      area_opacity :          0.5,
      plot_padding :          0,
      font_size :             11,
      vertical_label_unit :   "hr",
      colours :               {workload: '#4b80b6'},
      background_colour :     "#fff",
      label_colour :          "#444",
      grid_colour :           "#ccf",
      markers :               "value",
      meanline :              { 'stroke-width': '2px', stroke: '#000' },
      draw_axis :             false,
      labels :                ['mon', '', 'wed', '', 'fri', '', 'sun',],
      datalabels :            {workload:'My workload'},
    });

Area graph with multiple areas:

    var areagraph3 = new Ico.AreaGraph($('areagraph3'),
      {
      workload:       [8, 10, 6, 12,7, 6, 9],
      your_workload:  [6, 8,  4, 8, 12,6, 2],
      his_workload:   [2, 9,  12,7, 8, 9, 8]
      });

Area graph with multiple areas and custom options:

    var areagraph4 = new Ico.AreaGraph($('areagraph4'),
      {
      workload:       [8, 10, 6, 12,7, 6, 9],
      your_workload:  [6, 8,  4, 8, 12,6, 2],
      his_workload:   [2, 9,  12,7, 8, 9, 8]
      },
      {
        markers : "value",
        grid:false,
        draw_axis: false,
        plot_padding : 0,
        show_vertical_labels : false,
        show_horizontal_labels : false,
        hover_colour: "#000",
        hover_text_colour: "#fff",
        datalabels: {
          workload:       "My workload",
          your_workload:  "Your workload",
          his_workload:   "His workload"
        }
      });

Area graph options

markers
Line graph: display markers on the datapoint. (string)
Possible options:
  • "circle" - displays a circle
  • "value" - displays a hover with the value
Default value: false
marker_size
The size of the marker (number)
Default value: 5
hide_empty_label_grid
hide the vertical gridlines for all empty label on the horizontal axis. (boolean)
Default value: false
area_opacity
set a specific opacity for the area's. If none is given, single areas will be opaque, and multiple will get an opacity of 1.5/number-of-areas. (number between 0 and 1)
Default value: false

Stacked area graphs

Area graphs only support positive ranges.

Stack graph:

    var stackgraph = new Ico.StackGraph($('stackgraph'),
      {
      workload:       [8, 10, 6, 1,7, 6, 9],
      your_workload:  [6, 8,  4, 1, 12,6, 2],
      his_workload:   [2, 9,  12,2, 8, 9, 8]
      });

Stack graph with custom options:

    var stackgraph2 = new Ico.StackGraph($('stackgraph2'),
      {
      workload:       [8, 10, 6, 1,7, 6, 9],
      your_workload:  [6, 8,  4, 1, 12,6, 2],
      his_workload:   [2, 9,  12,2, 8, 9, 8]
      },
      {
        markers : "value",
        draw_axis: false,
        plot_padding : 0,
        hover_text_colour: "#fff",
        datalabels: {
          workload:       "My workload",
          your_workload:  "Your workload",
          his_workload:   "His workload"
        }
      });

Stack line graph:

    var stackgraph3 = new Ico.StackGraph($('stackgraph3'),
      {
      workload:       [8, 10, 6, 1,7, 6, 9],
      your_workload:  [6, 8,  4, 1, 12,6, 2],
      his_workload:   [2, 9,  12,2, 8, 9, 8]

        stacked_fill : false,
        markers : "value"
      });

Stacked area graph options

markers
Line graph: display markers on the datapoint. (string)
Possible options:
  • "circle" - displays a circle
  • "value" - displays a hover with the value
Default value: false
marker_size
The size of the marker (number)
Default value: 5
hide_empty_label_grid
hide the vertical gridlines for all empty label on the horizontal axis. (boolean)

stacked_fill
specify if the stacked areas should be filled area's or lines (boolean)

Bar graphs

Bar graph:

    var bargraph = new Ico.BarGraph($('bargraph'), [31, 12, 12, 10, 6, 4, 26]);

Bar graph with custom options:

     var bargraph2 = new Ico.BarGraph($('bargraph2'),
    [31, 12, 12, 10, 6, 4, 26],
    {
    labels: ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul"],
    colour: '#4b80b6',
    meanline: true,
    vertical_label_unit: "%",
    bargraph_lastcolour: "#666666",
    hover_colour : "#006677",
    datalabels: {one: ["January", "February", "March", "April", "May", "June", "July"]}
    });

Bar graph with negative values:

    var bargraph = new Ico.BarGraph($('bargraph3'),
    [31, 12, -12, -10, 6, -4, 26],
    {
     labels: ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul"],
     colour: '#4b80b6',
     meanline : true,
     hover_colour : "#006677",
     bargraph_negativecolour : "#660000",
     datalabels: {one: ["January", "February", "March", "April", "May", "June", "July"]}
    });

Bar graph options

bargraph_lastcolour
select a different colour for the last bar in a bargraph (string, example: "#fff")
Default value: false
bargraph_negativecolour
select a different colour for negative values in a bargraph (string, example: "#fff")
Default value: false
