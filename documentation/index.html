Ico graphing library Documentation

Ico is a graphing library based on Raphaël and Prototype.js. It is originally developed by Alex Young, but this version was mostly developed by Kilian Valkhof at Wakoopa. Ico's basic principles are to provide good, clean looking graphs inspired by work of Stephen Few and Edward Tufte, while still being flexible to implement and providing numerous api options to tweak the look of the graphs..

Some key things Ico supports:

  • Flexible ranges Ico select the best range to display your data at
  • Hover options display additional information such as the value when hovering over charts
  • Mean lines A single line that display the mean.
  • Watermarks Use an image as a watermark over your graphs

Basics

Ico graphs are initiated via this function call:

var foo = new Ico.GraphType($(element), data, options);

$(element) is a prototype id selector, data is an object or array (depending on the graph) and options is optional and an object.

Dependencies

  • Raphaël 1.0+
  • Prototype.js 1.6+

Supported graph types:

  • Line graphs
  • Area graphs
  • Stacked area graphs
  • Bar graphs
  • Horizontal bar graphs
  • Sparklines
  • Sparkbars
  • Sparkareas

Base graph options:

width
Specify the width of the element (number)
Default value: width of $(element)
height
Specify the height of the element (number)
Default value: height of $(element)
labels
The labels being displayed on the horizontal axis (array of strings)
Default value: numerical
grid
draw a grid or not (boolean)
Default value: true
plot_padding
Padding at the start and end of the graph (number)
Default value: 10
font_size
The font-size for the axis and hovers (number)
Default value: 10
show_vertical_labels
Show the vertical axis and gridlines (boolean)
Default value: true
show_horizontal_labels
Show the horizontal axis and gridline (boolean)
Default value: true
vertical_label_unit
append a unit to the vertical axis and text in the hover (string)
Default value: empty
colours
Select the colours for the lines or bars (object)
Default value: random colours
background_colour
The background colour of the graph (string, example: "#fff")
Default value: backgroundcolor of $(element), else #fff
label_colour
The colour of the axis, axis values and hovers (string, example: "#fff")
Default value: "#000"
grid_colour
The colour of the gridlines (string, example: "#fff")
Default value: "#ccc"
hover_text_colour
the colour of the text in the hovers (string, example: "#fff")
Default value: "#fff"
meanline
Display a line on the average of all datapoints. Accepts all Raphael parameters (object or boolean, example: { 'stroke-width': '2px', stroke: '#000' })
Default value: false, setting it to true will draw a 2px grey line.
padding_top
The amount of padding above the graph (number)
Default value: 20
draw_axis
show or hide the axis lines (boolean)
Default value: true
datalabels
for line graphs: Add a hover with the text, and append the text to the marker hover, per line (object, example: {workload:'My workload'})
for bar graphs: append a text per bar. (object, example: {workload: ["1","2","5"]})
Default value: empty
hover_colour
colour a bar or line should fade to on hover (string, example: "#fff")
Default value: empty
watermark
Add in an image to be used as watermark. (link to image or image element)
Default value: false
watermark_location
where the watermark should be placed. default is the bottom right, can be placed in the center with "middle" (string)
Default value: bottom right
left_padding
Set a standard left spacing regardless of vertical axis label width (number)
Default value: false

Line graphs

Line graphs accept ranges of negative and positive numbers.

Line graph with the default rendering (no options):

   var linegraph = new Ico.LineGraph($('linegraph'),{ workload: [8, 10, 6, 12, 7, 6, 9] });

Line graph with custom options:

var linegraph2 = new Ico.LineGraph($('linegraph'),
{ workload: [8, 10, 6, 12, 7, 6, 9] },
{
  grid :                  true,
  plot_padding :          0,
  stroke_width :          3,
  font_size :             11,
  vertical_label_unit :   "hr",
  colours :               {workload: '#4b80b6'},
  background_colour :     "#fff",
  label_colour :          "#444",
  grid_colour :           "#ccf",
  markers :               "value",
  meanline :              { 'stroke-width': '2px', stroke: '#000' },
  draw_axis :             false,
  labels :                ['mon', '', 'wed', '', 'fri', '', 'sun',],
  hide_empty_label_grid : true,
  datalabels :            {workload:'My workload'},
});

Line graph with multiple lines and a watermark:

var linegraph3 = new Ico.LineGraph($('linegraph3'),
{
  workload:       [8, 10, 6, 12,7, 6, 9],
  your_workload:  [6, 8,  4, 8, 12,6, 2],
  his_workload:   [2, 9,  12,7, 8, 9, 8]
},
{
  markers:            "circle",
  hover_colour:       "#000",
  hover_text_colour : "#fff",
  watermark : "Screenshot.png",
  watermark_location : "middle",
  datalabels: {
    workload:       "My workload",
    your_workload:  "Your workload",
    his_workload:   "His workload"
  }
});

Line graph with multiple lines and negative values:

var linegraph4 = new Ico.LineGraph($('linegraph4'),
{
  a : [8, -10, 6, 12,7, -6, -9],
  b : [6, 8,  -4, 8, -12,6, 2],
  c : [2, 9,  12,7, 8, -9, 8]
});

.

Line graph options:

markers
Line graph: display markers on the datapoint. (string)
Possible options:
  • "circle" - displays a circle
  • "value" - displays a hover with the value
Default value: false
marker_size
The size of the marker (number)
Default value: 5
hide_empty_label_grid
hide the vertical gridlines for all empty label on the horizontal axis. (boolean)
Default value: false
start_at_zero
If false, Ico will not start drawing untill the value of a line graph becomes more than zero. (boolean)
Default value: true
stroke_width
specify the width of the lines in pixels (number)
Default value: 5

Area graph options

markers
Line graph: display markers on the datapoint. (string)
Possible options:
  • "circle" - displays a circle
  • "value" - displays a hover with the value
Default value: false
marker_size
The size of the marker (number)
Default value: 5
hide_empty_label_grid
hide the vertical gridlines for all empty label on the horizontal axis. (boolean)
Default value: false
area_opacity
set a specific opacity for the area's. If none is given, single areas will be opaque, and multiple will get an opacity of 1.5/number-of-areas. (number between 0 and 1)
Default value: false
stroke_width
specify the width of the lines in pixels (number)
Default value: 0

Bar graphs

Bar graphs support posivite and negative ranges.

Bar graph:

var bargraph = new Ico.BarGraph($('bargraph'), [31, 12, 12, 10, 6, 4, 26]);

Bar graph with custom options:

var bargraph2 = new Ico.BarGraph($('bargraph2'),
[31, 12, 12, 10, 6, 4, 26],
{
  labels :              ["Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul"],
  colour :              '#4b80b6',
  meanline :            true,
  vertical_label_unit : "%",
  bargraph_lastcolour : "#666666",
  hover_colour :        "#006677",
  datalabels :          {one: ["January", "February", "March", "April", "May", "June", "July"]}
});

Bar graph with negative values:

var bargraph = new Ico.BarGraph($('bargraph3'),
[31, 12, -12, -10, 6, -4, 26],
{
  labels:                   "Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul"],
  colour:                   '#4b80b6',
  meanline :                true,
  hover_colour :            "#006677",
  bargraph_negativecolour : "#660000",
  datalabels :              {one: ["January", "February", "March", "April", "May", "June", "July"]}
});

Bar graph options

bargraph_lastcolour
select a different colour for the last bar in a bargraph (string, example: "#fff")
Default value: false
bargraph_negativecolour
select a different colour for negative values in a bargraph (string, example: "#fff")
Default value: false
colour
Select the colour for the bars (string)
Default value: random colour

Sparkline

Sparklines support posivite and negative ranges.

Sparkline: 

var sparkline = new Ico.SparkLine($('sparkline1'), [30, 12, 12, 10, 6, 4, 26]);

Sparkline with custom options: 

var sparkline2 = new Ico.SparkLine($('sparkline2'),
[30, 12, 12, 10, 6, 4, 26],
{
  stroke_width :      3,
  colour :            '#4b80b6',
  highlight :         {index: 2, colour: "#f00"},
  acceptable_range :  [0,20],
  background_colour : "#ddd"
});

Sparkline options

stroke_width
specify the width of the sparkline in pixels (number)
Default value: 1
highlight
draw a circle on one of the datapoints (object, example: {index: 2, colour: "#f00"})
colour is the color of the circle, index is the datapoint to be highlighted (zero-based)
Default value: false, setting it to true will draw a red dot on the last datapoint.
acceptable_range
draws the background between two value to denote an acceptable range (array, example: [0,10])
Default value: false, needs an explicit background colour
colour
Set the colour (string)
Default value: random colour

Sparkarea options

stroke_width
specify the width of the sparkline in pixels (number)
Default value: 1
highlight
draw a circle on one of the datapoints (object, example: {index: 2, colour: "#f00"})
colour is the color of the circle, index is the datapoint to be highlighted (zero-based)
Default value: false, setting it to true will draw a red dot on the last datapoint.
acceptable_range
draws the background between two value to denote an acceptable range (array, example: [0,10])
Default value: false, needs an explicit background colour
colour
Set the colour. If given an array, the first value will be used for the stroke and the second for the area. If given a string, it will be used for both, with the area being given 0.2 opacity. (string or array)
Default value: random colour
